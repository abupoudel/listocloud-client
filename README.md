# Listo Cloud Desktop Client

## Introduction

The Listo Cloud Desktop Client is a tool to synchronize files from Listo Cloud Server
with your computer.

## Download

### Binary packages

* Refer to the download page [LINK NOT MADE]

### Source code

The ownCloud Desktop Client is developed in Git. Since Git makes it easy to
fork and improve the source code and to adapt it to your need, many copies
can be found on the Internet, in particular on GitHub. However, the
authoritative repository maintained by the developers is located at
https://github.com/abupoudel/listocloud-client.

## Building the source code

[Building the Client](http://doc.owncloud.org/desktop/2.3/building.html)
in the ownCloud Desktop Client manual.
